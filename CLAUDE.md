# Progetto: SegretarIA
Istruzioni operative per Claude Code. Mantieni le risposte concise e ottimizza l'uso dei token.

## Stack Tecnologico
- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Lingua risposte: Italiano

## Linee Guida per il Codice & Ottimizzazione Token
- **Risposte sintetiche:** Non riscrivere interi file. Quando fai una modifica, mostra *solo* la funzione o il blocco di codice modificato.
- **No boilerplate:** Evita spiegazioni teoriche prolisse a meno che non ti venga esplicitamente richiesto.
- **Tailwind:** Usa le classi utility standard, evita stili inline o CSS personalizzato se non strettamente necessario.
